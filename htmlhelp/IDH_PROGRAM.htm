Launch

Start program

Enter the name and parameters of an X client you want to start and display on the X server. For example, to run xeyes 
xeyes -fg red

Start program on this computer

Start the chosen or entered program (with parameters as required) on the local machine.

Start program on remote computer

As above but the program will be run on a remote machine.

Using SSH

An X client program can be started remotely via SSH.

There is no mechanism to save a ssh password for security reasons.

SSH X11 forwarding must be enabled in the sshd daemon's sshd_config file on the remote machine. 

X11Forwarding yes
X11DisplayOffset 10
X11UseLocalhost yes

Prompt for password
Not yet implemented. ssh is run within a terminal, allowing the password has to be entered interactively.

Use ssh-agent or keychain
Not yet implemented. From bash you can start a ssh-agent daemon and run xlaunch using 
$ keychain id_rsa
$ . ~/.keychain/$HOSTNAME-sh
$ xlaunch -run my_ssh.xlaunch
Then XLaunch detects SSH_AGENT_PID and runs ssh without prompting for password.

When finished with ssh-agent 

$ keychain -k; unset SSH_AGENT_PID; unset SSH_AUTH_SOCK

Troubleshooting SSH
The command run automatically after the X server has started is 
ssh -Y <user@computer> <program>
Try this by hand, from a shell, if you have problems.
